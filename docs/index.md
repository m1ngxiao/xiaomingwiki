# Xiaoming Wiki

Xiaoming Wiki 是一个个人技术博客和个人 Wiki，用来长期记录学习过程、技术笔记、课程笔记、工程实践和阶段性复盘。

这里的核心目标不是堆课程清单，而是把输入资料沉淀为可以长期演进的知识系统。

## 五大知识方向

- [数学基础](./math/index.md)：微积分、线性代数、概率论、统计方法。
- [编程语言](./programming/index.md)：Python、C++、工具链、调试和工程实践。
- [计算机基础](./cs/index.md)：数据结构与算法、组成原理、操作系统、计算机网络。
- [AI 基础与模型](./ai-models/index.md)：机器学习、深度学习、CV、NLP、LLM、扩散模型。
- [AI 系统与性能](./ai-systems/index.md)：GPU、CUDA、算子优化、推理系统、量化和系统优化。

## 使用方法

建议先从栏目首页建立整体框架，再进入具体笔记。每篇笔记尽量回答三个问题：

1. 这个概念解决什么问题？
2. 它和前后知识有什么联系？
3. 能否通过推导、代码或实验验证？
