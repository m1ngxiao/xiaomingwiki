# 操作系统

操作系统提供进程、线程、虚拟内存、文件系统和 I/O 抽象，是系统工程的核心基础。

## 子方向

- 进程与线程。
- 调度。
- 同步与并发。
- 虚拟内存。
- 文件系统。
- I/O 与设备。

## 推荐学习路径

先理解进程和内存抽象，再学习并发、调度和文件系统，最后结合系统调用和性能分析观察真实行为。

## 笔记模板

- 操作系统抽象。
- 核心数据结构。
- 状态转换。
- 并发风险。
- 性能影响。
- 实验记录。
