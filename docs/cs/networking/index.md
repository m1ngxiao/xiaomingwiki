# 计算机网络

计算机网络解释数据如何跨机器传输，是理解分布式系统、服务部署和推理系统通信的基础。

## 子方向

- 分层模型。
- 链路层与网络层。
- TCP / UDP。
- HTTP / HTTPS。
- DNS、CDN 和负载均衡。
- RPC 与服务通信。

## 推荐学习路径

先理解分层和 TCP/IP，再学习 HTTP、DNS 和服务部署，最后结合分布式训练和推理服务理解通信瓶颈。
