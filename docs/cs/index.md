# 计算机基础

计算机基础基本对应 408 的核心知识体系，用来支撑后续系统工程和 AI Infra 学习。

## 子方向

- [数据结构与算法](./algorithms/index.md)
- [计算机组成原理](./computer-organization/index.md)
- [操作系统](./operating-systems/index.md)
- [计算机网络](./networking/index.md)

## 推荐学习路径

先学习数据结构和算法，建立复杂度意识；再学习组成原理和操作系统，理解程序如何运行；最后学习网络，理解分布式系统和推理服务的通信基础。

## 笔记模板

- 问题模型。
- 核心抽象。
- 关键机制。
- 时间 / 空间复杂度或性能指标。
- 常见边界条件。
- 与工程实践的联系。
