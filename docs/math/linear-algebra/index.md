# 线性代数

线性代数关注向量、矩阵、线性变换和高维空间结构，是现代机器学习和系统优化中最常用的数学语言之一。

## 核心主题

- 向量空间与基。
- 矩阵乘法与线性变换。
- 秩、零空间、列空间和行空间。
- 特征值、特征向量和谱分解。
- SVD、QR、LU 等矩阵分解。
- 投影、最小二乘和正交性。

## 推荐学习路径

先理解线性方程组和矩阵乘法，再进入向量空间、特征值和矩阵分解，最后连接到数值线性代数和深度学习中的张量计算。

## 笔记模板

- 定义：对象和运算是什么？
- 几何直觉：它在空间中做了什么？
- 代数性质：哪些等式和不变量重要？
- 数值问题：计算时有哪些稳定性问题？
- 应用：它如何出现在模型训练或系统实现中？

## 参考资料

可以参考 MIT 18.06 等线性代数公开课程，也可以结合数值线性代数教材理解工程计算中的误差和稳定性。
