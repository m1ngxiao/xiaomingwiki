# 概率论

概率论为不确定性建模提供语言，也是统计学习、生成模型和强化学习的重要基础。

## 核心主题

- 样本空间、事件和概率。
- 随机变量与分布。
- 期望、方差、协方差。
- 条件概率、贝叶斯公式和独立性。
- 大数定律与中心极限定理。
- 常见分布族和采样方法。

## 推荐学习路径

先掌握随机变量和分布，再学习条件概率与期望，最后进入极限定理、统计推断和机器学习中的概率模型。

## 笔记模板

- 随机对象是什么？
- 分布如何定义？
- 期望和方差如何计算？
- 条件结构是什么？
- 是否有可以模拟的数值实验？
