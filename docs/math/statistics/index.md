# 统计方法

统计方法连接数据、模型和结论，重点关注如何从有限样本中估计、判断和验证。

## 核心主题

- 参数估计与最大似然。
- 假设检验与 p-value。
- 置信区间与不确定性。
- 线性回归与广义线性模型。
- 抽样、偏差和方差。
- 实验设计与因果问题入口。

## 推荐学习路径

先理解估计和不确定性，再学习检验、回归和实验设计。遇到模型评估问题时，把统计假设和数据生成过程写清楚。

## 笔记模板

- 数据从哪里来？
- 假设是什么？
- 估计量是什么？
- 误差如何衡量？
- 结论在什么条件下成立？

## 参考资料

可以结合概率论、机器学习教材和真实实验记录学习统计方法，避免只记公式。
