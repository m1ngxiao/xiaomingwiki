# 机器学习

机器学习关注如何从数据中学习规律，并在未见样本上做出可靠预测。

## 栏目定位

这里记录监督学习、无监督学习、泛化、评估、优化和模型选择，是深度学习与大模型学习的前置基础。

## 子方向

- 监督学习。
- 无监督学习。
- 特征工程。
- 损失函数与优化。
- 泛化与正则化。
- 评估与交叉验证。

## 推荐学习路径

先理解线性模型、概率模型和损失函数，再进入核方法、树模型、集成方法和现代深度学习接口。

## 笔记模板

- 任务和数据。
- 模型假设。
- 训练目标。
- 泛化分析。
- 评估指标。
- 失败案例。

## 参考资料

可参考 CS229、统计学习教材和经典机器学习论文，但笔记按模型与问题类型组织。
