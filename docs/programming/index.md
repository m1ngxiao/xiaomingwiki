# 编程语言

编程语言栏目记录语言基础、工程实践、工具链、调试技巧和高性能计算相关能力。

## 子方向

- [Python](./python/index.md)：语言基础、类型系统、包管理、数值计算和工程实践。
- [C++](./cpp/index.md)：现代 C++、内存模型、模板、并发、性能优化和工程构建。

## 推荐学习路径

先用 Python 建立快速实验能力，再用 C++ 理解性能、内存和系统边界。AI 系统方向需要两者配合：Python 负责编排和实验，C++ / CUDA 负责关键性能路径。

## 笔记模板

- 使用场景。
- 核心语法或 API。
- 常见坑。
- 调试方法。
- 工程约束。
- 性能观察。

## 参考资料

优先参考官方文档、成熟工程项目和可复现实验。语言笔记应尽量连接真实工程问题，而不是停留在语法列表。
